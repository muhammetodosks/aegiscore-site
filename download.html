Installation Overview

üîß BATUR_INSTALL

Installation Script

The BATUR_INSTALL script handles the complete deployment of AegisCore on Windows systems. It performs the following operations:

  • System compatibility checks
  • Creates secure installation directory
  • Deploys core security modules
  • Configures default policies
  • Establishes failsafe mechanisms
  • Sets up monitoring services

Note: Requires administrator privileges for proper installation.

üßπ BATUR_CLEAN

Cleanup Script

The BATUR_CLEAN script provides complete removal of AegisCore from the system. It ensures clean uninstallation without leaving residual components:

  • Stops all AegisCore services
  • Removes security policies
  • Deletes monitoring components
  • Cleans registry entries
  • Restores original system settings
  • Removes installation files

Note: System restart may be required for complete cleanup.
